## Selected Projects

### Real Estate Management Platform
Developer operations system covering land acquisition, permits, construction progress, procurement, dashboards, and real-time status monitoring.

**Stack:** Next.js, Express, TypeScript, PostgreSQL, SSE, GCP

### Amani Supplier
Internal material management and admin platform for construction inventory, product catalog maintenance, contractor order tracking, and procurement visibility.

**Stack:** Next.js, TypeScript, NestJS, PostgreSQL, Material UI, SSE

### HappyHomes CRM
Property sales CRM with customer tracking, order management, transaction dashboards, and real-time payment monitoring.

**Stack:** React, Express, TanStack Query, SSE, PocketBase, GCP

### QC Mobile
React Native mobile app for construction quality-control checks, photo verification, and on-site inspection workflows.

**Stack:** React Native, TypeScript, Express, PostgreSQL, GCP
